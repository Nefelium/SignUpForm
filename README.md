Form with Validation. Swift 4 with RxSwift and MVVM pattern.

Sign Up form with Validation:
- Name: more 2 symbols
- E-mail: real valid e-mail
- Phone: is filled with numbers

Used:
- MVVM pattern
- RxSwift
- RxCocoa
- AKMaskField pod
- IQKeyboardManagerSwift pod
- SwiftValidators
- TableView with dynamic cells
- IBInspectable
